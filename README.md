# CSS Practice Repository

A collection of HTML files demonstrating various CSS concepts and techniques.

## Topics Covered

| File | Topic |
|------|-------|
| `color.html` | CSS colors — named colors, hex, RGB, HSL |
| `Class.html` | CSS class selectors and specificity |
| `boxmodel.html` | Box model — margin, border, padding, content |
| `button.html` | Styling buttons with CSS |
| `list.html` | Styling ordered and unordered lists |
| `textalign.html` | Text alignment and typography |
| `Flexbox.html` | Flexbox layout — flex containers and items |
| `Card.html` | Building card components with CSS |
| `Navigation.html` | Styling navigation bars and menus |

## Getting Started

1. Clone the repository:
```bash
   git clone https://github.com/rvr-likitha6/CSS.git
```
2. Open any `.html` file in your browser to see the output.

## Prerequisites

No dependencies or installations needed — just a web browser.

## Purpose

This repository is meant for learning and practicing CSS fundamentals through hands-on examples.
